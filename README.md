# KEY FEATURES
1. User Authentication System.
2. User follow/unfollow functionalities.
3. Users search.
4. Post,Comment and Reply CRUD functionalities.
5. Profile CRUD functionalities and dashbaord.
6. Developers and profile display.

# KEY TECHNOLOGIES
1. Mongoose
2. Express
3. React
4. NodeJS
5. Redux
6. JSON Web Token
7. Bcrypt
8. Express Validator
9. Normalize url
10. React toastify
11. Timeago.JS
